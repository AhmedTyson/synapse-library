## <img src="https://api.iconify.design/lucide:library.svg?color=%238A2BE2" width="20" height="20" align="center" /> About

---

> "A centralized knowledge hub for coding mastery and bilingual technical documentation."

**Synapse Library** is an elegant, single-page documentation application operating on a purely dynamic JSON architecture. It features persistent progress tracking, syntax highlighting, and modular curriculum integration designed to make reading extensive documentation visually satisfying and friction-free.

## <img src="https://api.iconify.design/lucide:star.svg?color=%238A2BE2" width="20" height="20" align="center" /> Key Features

- <img src="https://api.iconify.design/lucide:check-circle-2.svg?color=%238A2BE2" width="16" height="16" align="center" /> **Persistent Progress Tracking**: Visual completion tracking built natively with LocalStorage.
- <img src="https://api.iconify.design/lucide:languages.svg?color=%238A2BE2" width="16" height="16" align="center" /> **Bilingual Content**: Intermixed Arabic/English explanations leveraging automated backticks parsing.
- <img src="https://api.iconify.design/lucide:database.svg?color=%238A2BE2" width="16" height="16" align="center" /> **Dynamic JSON Architecture**: The entire curriculum (OOP, PHP, JavaScript) is loaded asynchronously via granular JSON chunks.
- <img src="https://api.iconify.design/lucide:printer.svg?color=%238A2BE2" width="16" height="16" align="center" /> **PDF Export**: Integrated `html2pdf.js` capability allowing users to directly compile and save chapters offline.
- <img src="https://api.iconify.design/lucide:sparkles.svg?color=%238A2BE2" width="16" height="16" align="center" /> **Aesthetic Interface**: Prism.js-powered syntax highlighting wrapped in a dark-mode inspired UI blueprint.
